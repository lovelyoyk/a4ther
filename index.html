iPhone / iPad
Sysdiagnose (.tar.gz) — recomendado, sem JB
  1. Aperte simultaneamente: Volume + + Volume - + Botão Lateral (1 segundo)
  2. Sinta uma vibração leve confirmando · continue usando o iPhone normalmente
  3. Aguarde ~5 minutos (o iOS está gerando o dump)
  4. Vá em Ajustes → Privacidade → Análise & Melhorias → Dados de Análise
  5. Procure sysdiagnose_AAAA.MM.DD_*.tar.gz (mais recente)
  6. Toque no arquivo → Compartilhar → envie via AirDrop / Files / iCloud / WhatsApp
  7. Suba aqui pelo modo "iOS · sysdiagnose .tar.gz"
⚡ Se travar no upload: extraia localmente no iPhone (Files → toque longo no .tar.gz → Uncompress) e use o modo "iOS · sysdiagnose (pasta)".

Android
Via Termux (sem root) — gera .txt
  1. Instale o Termux pelo F-Droid (NÃO use o do Play Store — abandonado): f-droid.org/packages/com.termux/
  2. Abra o Termux e cole esse comando único:
pkg update -y && pkg install -y curl && rm -f a4ther.sh && curl -L -o a4ther.sh https://raw.githubusercontent.com/lovelyoyk/a4ther/main/a4ther.sh && chmod +x a4ther.sh && sh a4ther.sh
  1. Aguarde o scan terminar (~30-60s)
  2. O .txt salva em ~/a4ther_reports/scan_AAAAMMDD_HHMMSS.txt (= /data/data/com.termux/files/home/a4ther_reports/)
  3. Pra compartilhar do Termux pro WhatsApp/Telegram, rode:
cp ~/a4ther_reports/scan_*.txt /sdcard/Download/ && termux-open /sdcard/Download/$(ls -t ~/a4ther_reports/ | head -1)
  1. Abre o picker do Android — escolhe WhatsApp/Telegram/iCloud — manda pra cá
  2. Faz upload aqui pelo modo "Android · scan TXT"
📋 Atalho: termux-share também funciona (precisa app Termux:API). Sem ele, o cp pra /sdcard/Download/ + abrir Files já permite mandar pro WhatsApp.
📦 Source open: github.com/lovelyoyk/a4ther
